# Конструктор бюджета — пакет для публикации

Это статическая страница (один файл `index.html`), без сборки и без сервера — можно опубликовать на любом хостинге.

## Как опубликовать

**Вариант 1 — Netlify / Vercel (проще всего, бесплатно)**
1. Перейдите на netlify.com (или vercel.com), зарегистрируйтесь.
2. Перетащите эту папку в окно загрузки («Deploy manually» / «Drag and drop your project folder»).
3. Через несколько секунд получите ссылку вида `https://ваш-проект.netlify.app`.

**Вариант 2 — на свой сайт по FTP**
Загрузите `index.html` в нужную папку на вашем хостинге (например, в `/budget-tool/`) через FTP или панель управления хостингом. Страница будет доступна по адресу вида `https://вашсайт.ру/budget-tool/`.

**Вариант 3 — GitHub Pages**
Создайте репозиторий, положите туда `index.html`, включите GitHub Pages в настройках репозитория (Settings → Pages).

## Важно

- Страница подключает шрифты с Google Fonts через интернет — для показа клиентам/коллегам нужен доступ в интернет (это не локальное приложение).
- Внутри файла, в начале `<script>`, есть блок `PRICING` — числа для расчёта. Перед публикацией на сайт замените рыночные ориентиры на ваши реальные внутренние ставки, если хотите показывать инструмент клиентам как часть продаж, а не только использовать внутри команды.
- Если хотите давать ссылку именно на квиз отдельно (без конструктора), это отдельный файл `kviz_cenoobrazovanie.html`, который вам уже отправляли ранее — его можно упаковать так же.
